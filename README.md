# Flask Приложение - Случайные Вдохновляющие Цитаты

Веб-приложение на Flask с Bootstrap для получения и отображения случайных вдохновляющих цитат из публичных API.

## Возможности

- Случайный выбор одного из двух проверенных API для получения цитат:
  - **Forismatic** (https://api.forismatic.com/api/1.0/) - цитаты на русском языке
  - **FavQs** (https://favqs.com/api/qotd) - цитата дня на английском языке
- Красивый интерфейс с Bootstrap 5
- Автоматическое обновление цитат без перезагрузки страницы
- Обработка ошибок при недоступности API
- Все API проверены и работают без ключей доступа

## Установка

1. Активируйте виртуальное окружение:
```powershell
.\venv\Scripts\Activate.ps1
```

2. Установите зависимости:
```powershell
pip install -r requirements.txt
```

## Запуск

Запустите приложение:
```powershell
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## Использование

1. Откройте браузер и перейдите на http://localhost:5000
2. На странице автоматически отобразится случайная вдохновляющая цитата
3. Нажмите кнопку "Новая цитата" для получения следующей цитаты
4. Приложение автоматически выберет случайный API из доступных (Forismatic или FavQs)
5. Цитаты могут быть на русском (Forismatic) или английском (FavQs) языке
